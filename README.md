# LinguaMethod Admin Panel

Админ-панель для управления учебными материалами LinguaMethod — B2B SaaS платформы для преподавателей иностранных языков.

## Архитектура

### Учебник (теория)
Иерархия: **Курс → Модуль → Урок → Раздел → Блоки контента**

10 типов контент-блоков: текст, картинка, аудио, YouTube, разделитель, HTML, карточка слова, грамматика, диалог, тоновый блок.

### Банк упражнений + Рабочая тетрадь
Упражнения хранятся в **банке**, привязаны к уроку.

11 типов упражнений:
- **Автопроверка (8):** соединить пары, выбор ответа, заполнить пропуск, расставить тоны, порядок слов, грамматический выбор, перевод на китайский, перевод на английский
- **Ручная проверка (3):** диктант, описание картинки, свободное письмо

**Рабочая тетрадь** — набор упражнений из банка, назначенных на урок «по умолчанию». Вкладка «Тетрадь» в редакторе урока.

## Технологии
- Next.js 15 (App Router)
- TypeScript, Prisma + PostgreSQL
- Tailwind CSS 4, shadcn/ui
- NextAuth v5, Tiptap, KaTeX
- Vercel Blob (файлы)

## Запуск

```bash
npm install
cp .env.example .env  # Настроить DATABASE_URL
npm run db:push        # Создать таблицы
npm run db:seed        # Демо-данные
npm run dev            # http://localhost:3000
```
