# LinguaMethod Admin Panel

Админ-панель для управления учебными материалами LinguaMethod — B2B SaaS платформы для преподавателей иностранных языков.

## Архитектура

### Учебник (теория)
Иерархия: **Курс → Unit → Урок → Раздел → Блоки контента**

9 типов контент-блоков:
- **Базовые:** текст (Tiptap WYSIWYG), картинка, аудио, YouTube, разделитель, отступ (spacer), HTML-вставка
- **Языковые:** универсальная карточка слова, диалог

### Банк упражнений + Рабочая тетрадь
Упражнения хранятся в **банке**, привязаны к разделу (Section).

10 типов упражнений:
- **Автопроверка (4):** соединить пары, выбор ответа, расставить тоны, порядок слов
- **Ручная проверка (6):** заполнить пропуск, перевод, написать транскрипцию, диктант, описание картинки, свободное письмо

**Рабочая тетрадь** — упражнения с `isDefaultInWorkbook=true`.
**Банк** — упражнения с `isDefaultInWorkbook=false` (запас).
Упражнение может быть либо в тетради, либо в банке.
Можно создавать упражнения прямо из тетради.

## Технологии
- Next.js 15 (App Router)
- TypeScript, Prisma + PostgreSQL
- Tailwind CSS 4, shadcn/ui
- NextAuth v5, Tiptap (WYSIWYG)
- Vercel Blob (файлы)

## Запуск

```bash
npm install
cp .env.example .env  # Настроить DATABASE_URL
npm run db:push        # Создать таблицы
npm run db:seed        # Демо-данные
npm run dev            # http://localhost:3000
```
